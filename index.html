FAQ

Common questions.

Which OMOP CDM version does IVe target? expand_more

IVe targets OMOP CDM v5.4. Vocabulary version v20231012 is bundled by default and can be swapped out per deployment.

Does the Clinical Agent see PHI? expand_more

The Agent runs against the same OMOP database your account is authorized for. Access is gated by JWT-authenticated sessions and feature-level RBAC.

Can I extend IVe with my own widgets? expand_more

Yes. Widgets are React components registered against the workspace widget registry. New endpoint-backed widgets can be added via the Endpoint editor without redeploys.

How do I get access? expand_more

Sign in to the Hulab Apps suite, then request access to IVe from your Profile → Apps panel.
